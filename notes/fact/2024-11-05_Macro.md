# Macro

In Rust, you can see many exclamations marks (`!`) in the code. Such as:

```rust
println!("Hello, world!");
//     ^
```

```rust
let mut guess = vec![];
//                 ^
```

You can just treat them as regular functions right now. We will explain them in detail later.
