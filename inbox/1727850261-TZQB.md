# 掛載

掛載格式化分區以準備安裝：

```bash
mount /dev/sdXY /mnt # 掛載根分區
mkdir /mnt/boot
mount /dev/sdXY /mnt/boot # 掛載 EFI 分區

mkdir /mnt/home # 如果有 home 分區
mount /dev/sdXY /mnt/home # 如果有 home 分區

swapon /dev/sdXY # 如果有 swap 分區
```

sdXY 是您的分區名稱，例如 sda1，請根據實際情況替換。
