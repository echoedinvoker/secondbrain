|----------|--------------------------------------------------|

| `*`      | Multiplication                                   |
| `/`      | Division                                         |
| `^`      | Exponentiation                                   |
| `@`      | Absolute value                                   |
| `%`      | Modulus (remainder of division)                  |
| `|/`   | Square root                                      |
